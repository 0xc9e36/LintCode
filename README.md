## 起因
&emsp;&emsp;面试官让我手写一个类似字符串匹配的东西（kmp、sumday，bm），墨迹了好半天搞不出， 这时场面就变得十分尴尬了，相对无言。 明明都是烂熟的东西了，为什么一段时间不用就全忘光了，还是基础不够扎实， 这对于立志成为工程师的我，是很羞愧的，于是我开始了刷题的道路，以此记录。
