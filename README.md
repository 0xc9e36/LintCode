## 起因
    去面试的时候面试官让我手写一个类似字符串匹配的东西（kmp、sumday，bm），墨迹了好半天写不出， 这时场面就变得十分尴尬了。 明明都是烂熟的东西了，一段时间不用就全忘光了，所以我决定刷一刷算法题吧，虽然以后可能还会忘记，但至少比什么都不做有意义一些。
